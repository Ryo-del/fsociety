# Frontend Project for Hackathon

## Описание
Это небольшое веб‑приложение, реализующее формы входа и регистрации пользователей. Проект предназначен для демонстрации базовых возможностей UI/UX, а также взаимодействия с бекендом (в дальнейшем можно подключить API).

## Структура проекта
```
frontend/
├── auth/                # Страница входа/регистрации
│   ├── script.js        # Логика переключения форм и валидация
│   └── style.css        # Стили для форм
├── Create/              # Страница создания чего‑то (заготовка)
│   ├── create.html
│   ├── create.css
│   └── create.js
├── main/                # Основная страница приложения
│   ├── main.html
│   ├── main.css
│   └── main.js
├── myjob/               # Страница «Моя работа»
│   ├── myjob.html
│   ├── myjob.css
│   └── myjob.js
└── index.html           # Точка входа в приложение
```

## Как запустить
1. Убедитесь, что у вас установлен любой современный браузер.
2. Откройте файл `frontend/index.html` в браузере (двойной клик или `open frontend/index.html` в терминале).
3. При необходимости запустите локальный сервер (например, `python -m http.server 8000` в корне проекта) и откройте `http://localhost:8000/frontend/index.html`.

## Функциональность
- **Вход** – форма с полями «Логин» и «Пароль». Для теста используйте `login: user`, `password: 123456`.
- **Регистрация** – форма с полями «Логин», «Пароль», «Email».
- Переключение между формами реализовано через JavaScript (`frontend/auth/script.js`).
- Иконки Font Awesome подключены через CDN.

## Требования
- HTML5, CSS3, JavaScript (ES6+).
- Подключение к внешнему CDN для Font Awesome (см. `index.html`).
